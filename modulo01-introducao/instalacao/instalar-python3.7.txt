## Instalar Python 3.7 ##

## Comandos utilizados, ubuntu 18.04 lts. ##

sudo apt install python3.7 python3.7-dev python3-venv python3.7-venv python3-venv idle-python3.7 python3-pip virtualenv
python3.7 -m pip install --upgrade pip setuptools wheel --user
python3.7 -m pip install pipenv --user
export PATH="/home/usuario/.local/bin:$PATH"

## Instalar Pycharm Community ##

No ubuntu basta ir a loja de aplicativos, procurar por Pycharm, e executar alguns comandos.

## Instalar Pycharm Community via snap ##

sudo apt install snapd snapd-xdg-open
sudo snap install pycharm-community --classic
